
# ToDo-List

A web application platform that allows you to save your daily tasks so as not to miss anything.
## Features
Pick a date from callendar
Add new task
Mark task as completed / uncompleted
Set task priority
Edit task
Delete single task
Delete completed tasks
Delete all tasks

## Technologies used:
HTML
CSS
JavaScript

## Demo

  npm run deploy
